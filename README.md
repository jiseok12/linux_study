# linux

## 01

### ls
    'ls' 현재 디렉토리의 파일을 보여 준다
    'ls -l' 현재 디렉토리의 파일 목록을 자세히 출력하는 명령어. 
    'ls -al' 은 현재 디렉토리의 모든 파일 목록을 출력한다

### pwd
    현재 위치하고 있는 디렉토리를 알려주는 명령어
    'pwd'

### mkdir
    디렉토리를 생성 한다
    'mkdir 생성할 디렉토리 명'

## 02

### cd
    디렉토리 이동
    cd 이동할 디렉토리의 경로
    'cd ..'  현재 디렉토리의 부모 디렉토리로 이동할 경로
    'cd /이동할 디렉토리의 경로' '/' <- 최상의 디렉토리로 부터 이동할 경로

#### 상대경로
    '..' 현재 디렉토리의 위치를 기준으로 다른 디렉토리의 위치를 표현하는 것
#### 절대 경로
    '/' 최상위 디렉토리를 기준으로 경로를 표현하는 것(최상위 디렉토리를 루트'root'디렉토리라고 한다)

### rm
    디렉토리를 삭제한다
    'rm 삭제할 파일명'
    'rm -r 삭제할 파일명' 디렉토리 안에있는 다른 디렉토리까지 삭제한다

### --help
    명령어 뒤에 --help를 붙이면 명령어의 사용설명서가 출력된다
    예) ls --help
    '명령어 --help'
